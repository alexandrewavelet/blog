Blog – PHP

TP de création d'un blog en PHP.
technologies : PHP, SQL, Smarty, Bootstrap

FONCTIONNALITES
- Affichage des articles avec pagination
- Création/Modification/Supression d'articles
- Système d'identification avec sessions et cookies
- Utilisation de Smarty (Framework MVC en PHP)
- Système de recherche d'articles
- Upload d'images JPEG, et redimensionnement (200px large)
- JS pour vérifier que tous les champs sont remplis (création d'article)
- Système de tag

INSTALLATION
- importer le fichier blog.sql dans votre BDD
- modifier le fichier includes/connexion.inc.php avec vos informations de connexion
- Pour se connecter, les identifiants sont "admin" et "azerty" en mot de passe
